# Electronics Development

This folder contains electronics prototyping and versioning documentation for the JJC club project.

## Goal

- Prototype LED control and driver electronics for future club versions
- Support WS2812B / NeoPixel-style addressable LEDs
- Keep early tests on the bench before integrating electronics into a printed club
- Provide a path toward motion sensing, timing, club sync, and app control

## Structure

- `v1/` - initial ESP32 and LED prototype docs
- `v2/` - planned future refinements and physical integration notes
- `README.md` - overview and compatibility notes

## Recommended Docs

- `v1/parts_list.md` - ESP32-based parts and prototyping items
- `v1/assembly_guide.md` - step-by-step ESP32 and LED assembly guide
- `v1/esp32_programming.md` - Arduino IDE and MicroPython programming notes
