# V3 Handle Sleeve Spec

## Purpose

The handle sleeve creates the long grip around the dowel.

V3 keeps the V2 handle sleeve geometry.

Reference diagram:

```text
Resources/Images/3_handle_sleeve_fit.svg
```

## Geometry

```text
Knob-side outer diameter: 18.2 mm
Grip/collar-side outer diameter: 24 mm
Inner diameter: 13.2 mm
Length: 185 mm
Side profile: single straight taper over full length
```

## Balance Role

The handle is the main lower-mass region used to keep the club from feeling nose-heavy.

Estimated contribution:

```text
Mass: 47.7 g
Center: 134.3 mm from knob end
```

## Print Orientation

Print vertically if TPU is stable enough. If the print wobbles, split the handle into two shorter sleeve sections.
