# V3 Handle-to-Body Collar Spec

## Purpose

The collar transitions from the slim handle to the hollow body shell.

V3 keeps the V2 collar geometry so the handle/body layout stays compatible.

Reference diagram:

```text
Resources/Images/4_handle_to_body_collar_fit.svg
```

## Geometry

```text
Handle-side outer diameter: 24 mm
Body-side outer diameter: 42 mm
Inner diameter: 13.2 mm
Height: 20 mm
Side profile: straight taper
```

## Balance Role

The collar sits close to the V3 target balance region and is not a major tuning part.

Estimated contribution:

```text
Mass: 17.8 g
Center: 225.1 mm from knob end
```

## Print Orientation

Print either end down. Body-side down gives the larger bed contact area.
