# V2 Handle Sleeve Spec

## Purpose

The handle sleeve creates the long Delphin-style grip around the dowel.

## Geometry

```text
Knob-side outer diameter: 18.2 mm
Grip/collar-side outer diameter: 24 mm
Inner diameter: 13.2 mm
Length: 185 mm
Side profile: single straight taper over full length
```

## Print Orientation

Print vertically if TPU is stable enough. If the print wobbles, split the handle into two shorter sleeve sections later.
