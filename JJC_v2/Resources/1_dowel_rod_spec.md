# V2 Dowel Rod Spec

## Purpose

The dowel rod is the rigid center spine for the JJC V2 Delphin-style club.

Reference diagram:

```text
Resources/Images/1_dowel_rod_fit.svg
```

## Geometry

```text
Diameter: 12.7 mm
Length: 502 mm
Material: hardwood dowel
```

## Notes

Measure the actual dowel before finalizing printed socket clearances. Printed TPU parts use a starting hole/socket diameter of 13.2 mm.
